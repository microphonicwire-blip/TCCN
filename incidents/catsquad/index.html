Event E002 — The Cat Squad Bombing

🚨 SERVER RAID ⚔️ CONFLICT 🕵️ CONSPIRACY RESOLVED

Summary​

Beginning around March 2025, Audon was invited by himself to bring Federal Gamers members into an external server called Cat Squad, owned by Charlie (Fluffy). Initially the server was peaceful, however Audon grew to dislike Cat Squad over time due to its immature culture and the negative influence he felt it had on his friend, Easy. Audon also developed a personal conflict with a member known only as PoopCat, writing large diss paragraphs — later found to be plagiarised from content creators Packgod and Koofy.

This tension eventually escalated into a coordinated raid and destruction of the Cat Squad server.

Method​

The raid was executed by socially engineering Cat Squad members into granting Audon admin privileges. Once obtained, Audon:

  • Pinged @everyone
  • Deleted every channel in the server live, in front of its members
  • Left a single channel titled "Bombed by Federal Gamers"

Aftermath​

  • Charlie reportedly cried during the raid
  • Cat Squad members sent death threats to Audon and Josh
  • Cat Squad restored itself using pre-prepared backup servers
  • Audon expressed no remorse, stating the raid was done "for Easy"

Ongoing Concerns​

Active Threat

Following the raid, Audon created a server called FG Bureaucracy, reportedly used to build cases against individuals and store harvested IP addresses. The location of this server and its data is currently unknown.

Notes​

  • The diss paragraphs Audon wrote against PoopCat were later found to be plagiarised from content creators Packgod and Koofy
  • Josh showed hesitation during the raid but did not take action to stop it
  • This event demonstrates a pattern of coordinated server raiding that may continue
