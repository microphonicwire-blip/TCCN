# 🚨 Incidents

This section contains all logged criminal incidents on the server. Each incident is documented with a unique ID, date, severity, and any involved parties.

## How to Add an Incident

To submit a new incident for logging, post in the appropriate server channel with the following format:

```
Incident Date: DD/MM/YYYY
Severity: Low / Medium / High
Description: [What happened]
Involved: [Discord usernames]
Evidence: [Screenshots / links]
```
